# Product Brief — Web Factory Agency

## ¿Qué hace este negocio?
Agencia de desarrollo web de una persona en RD que usa inteligencia artificial para investigar prospectos, generar propuestas personalizadas y entregar sitios web profesionales en 7 días — más rápido y más barato que cualquier agencia tradicional.

## ¿Para quién es?
Empresas dominicanas medianas (20-100 empleados) con web básica o sin web: colegios, clínicas, constructoras, distribuidoras. El decisor es el dueño o gerente general. Presupuesto: RD$15,000–40,000 por proyecto.

## Tech Stack
- Frontend: HTML/CSS/JS o WordPress según cliente
- Backend: Supabase (portal interno)
- Research: Python daemon + Claude API
- Portal: GitLab Pages (estático)
- Hosting cliente: Vercel / Netlify / cPanel

## Ventaja competitiva
El portal investiga la empresa del prospecto antes de la llamada: web actual, redes sociales, Google Maps, competencia. La propuesta llega personalizada el mismo día del primer contacto.

## Meta del sprint actual
Cerrar 3 contratos en los próximos 14 días.
Target mínimo: RD$15,000 × 3 = RD$45,000
